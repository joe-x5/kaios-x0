The way this calculator works is based on mathematical expressions. Some of this expressions looks like this:

3+98(-52) 5x9+√(25) -109÷5! sin((()))

The last example It makes no sense and not display anything, this happen with all the expressions they are not well constructed. You can insert the last result in the current expression with the 'Ans' option on the menu.

To enter the mathematical operators, pressing the down key or the up key the selector will move changing color in the boxes between the mathematical operators DEL, ÷, x, -, + once you have selected the desired mathematical operator, press the Enter key to enter it on screen.

Remember you only can press = and show the result of the operation only when the Scientific panel is close.

Scientific panel
